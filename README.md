# UIColorHex

Define UIColor by Hex

## Example Code 
### UIColor (UIKit)
```UIColor.swift
// without alpha
let color: UIColor = UIColor(hex: "000000")

// 透明度指定 (alpha)
let color: UIColor = UIColor(hex: "000000", alpha: 0.5)
```
### Color (SwiftUI)
```Color.swift
// without alpha
let color: Color = Color(UIColor(hex: "000000"))

// 透明度指定 (alpha)
let color: Color = Color(UIColor(hex: "000000", alpha: 0.5))
```
